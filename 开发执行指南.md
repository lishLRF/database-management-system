#### 第3步：项目脚手架搭建（1天）

**执行任务**: Task #1（见 `docs/superpowers/tasks/implementation-tasks.md`）

**后端脚手架**:
```bash
# 使用 Spring Initializr 生成项目
# https://start.spring.io/
# 参数:
# - Project: Maven
# - Language: Java
# - Spring Boot: 3.2.5
# - Java: 21
# - Dependencies: Spring Web, Spring Security, Spring JDBC, Validation

# 或使用命令行生成
curl https://start.spring.io/starter.zip \
  -d dependencies=web,security,jdbc,validation \
  -d type=maven-project \
  -d javaVersion=21 \
  -d bootVersion=3.2.5 \
  -d baseDir=backend \
  -o backend.zip && unzip backend.zip
```

**前端脚手架**:
```bash
# 创建 Vue 3 + TypeScript 项目
npm create vite@latest frontend -- --template vue-ts

cd frontend
npm install element-plus axios pinia vue-router
npm install --save-dev @types/node
```

#### 第5步：持续集成配置（0.5天）

**创建 CI/CD 配置**:

```yaml
# .gitlab-ci.yml 或 .github/workflows/ci.yml

stages:
  - build
  - test
  - deploy

backend-build:
  stage: build
  script:
    - cd backend
    - mvn clean package -DskipTests
  artifacts:
    paths:
      - backend/target/*.jar

frontend-build:
  stage: build
  script:
    - cd frontend
    - npm install
    - npm run build
  artifacts:
    paths:
      - frontend/dist/

backend-test:
  stage: test
  script:
    - cd backend
    - mvn test

frontend-test:
  stage: test
  script:
    - cd frontend
    - npm run test
```

## ⚠️ 风险预警

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| AI API不稳定 | 核心功能不可用 | 增加重试机制 + 降级为手动模式 |
| 多数据源并发问题 | 数据错乱 | 严格测试ThreadLocal隔离 |
| 大结果集性能 | 用户体验差 | 强制分页 + 虚拟滚动 |
| 安全漏洞 | 数据泄露 | SQL注入测试 + 敏感数据加密 |
